# Converter 
A web interface for downloading YouTube videos. Built with **Bootstrap 5** and **JavaScript**.

## About The Project

This repository contains the **frontend source code** for a personal project.

**Features:**
* **Format Selection:** Switch between MP3 (Audio) and MP4 (Video).
* **Quality Control:** Select 1080p, 720p, or Audio Bitrates.


## How to Use 

1. Paste the Youtube Link on the URL input box.
2. Select Which Format and Quality to use.
3. Convert & Download, Wait for it to finish. As it will automatically download.

Note:
This doesn't support Playlists.
